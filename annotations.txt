// Coach-Luiz tip:
// É mais vantajoso e menos complexo - ao migrar uma aplicação de '.js' para '.ts'
// - fazer essa migração de dentro para fora, ou seja, o dev começa pelo código
// mais interno (ou componente mais interno) e depois vai "tipando" os componentes
// mais externos que fazem uso desses componentes mais internos. Isso evita retrabalho.
